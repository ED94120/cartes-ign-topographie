# cartes-ign-topographie

L'application extrait une image géoréférencée des cartes IGN avec différents overlays

Un des overlays donne la position des antennes-relais en mars 2026 (document ANFR open data) 

Fichier contenant la position des supports des antenens : "./data/GeoJSON-support_antennes_ANFR-light-2026-04-03.geojson"
